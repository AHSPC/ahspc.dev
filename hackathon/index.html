AHS Hackathon 2023!

A hackathon is a programming competition where participants make and submit an original project. AHS Programming Club is running a school wide hackathon from April 19th to April 26th!

What do you do in it?

1. You think of an idea for some software project.
2. Between the start and end dates (April 19-26) you make it, and submit it.
3. Winners will be announced soon after.

How do I participate?

1. Read through this webpage. (Great, you're doing that already!)
2. Fill out the sign-up form: https://forms.gle/bRwdvMfRufsjYcULA
3. Join the discord: https://discord.gg/aSgKFxqxTr and find the #hackathon-announcements channel. Instructions to submit will be posted there soon!

Prizes

We'll have up to $100 in prizes, split across 3 categories:

  • Best Overall (idea + project, project has more weight)
  • Best Idea (idea only)
  • Undefined Prize (don't ask otherwise you'll cause an error)
